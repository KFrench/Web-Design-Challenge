Latitude Analysis Dashboard

The purpose of this assignment was to use data to determine the impact of distance from the equator to different atmospheric elements. Each visualization page includes a graphical representation of latitude versus four different measures. Those measures are: maximum temperature, humidity, cloudiness, and wind speed. Using the OpenWeatherMap API, a python script was created to visualize the weather of 500+ cities around the world located at varying distances from the equator. The Pandas Matplotlib module was then used to build multiple plots displaying the results of the python script.
